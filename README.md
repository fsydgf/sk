　　明明是正午时分，却好似到了夜间。
　　雷电伴随着打雷声在云层中酝酿，仰头看天，甚至能看到银蛇在云层中游动。
　　“不会吧，今天出来得这么不凑巧？”
　　李念凡站在山林之中，看着天空苦笑的摇了摇头。
　　他今天来了兴致，便带着大黑在山林里转悠，顺便看看能不能打些野味，怎么说下雨就要下雨了。
　　“啪嗒啪嗒。”
　　短短几分钟，已经有豆大的雨水从天空滴落而下。
　　“得赶紧找个地方躲雨了。”李念凡迅速扫视着周围的环境。
　　“汪汪汪！”
　　大黑冲着一个方向叫唤了几声，便迈着四肢奔跑而去。
　　跟着大黑，李念凡很快就看到了一个山洞。
　　“运气不错，这居然是个空山洞，大黑，好样的！”
　　李念凡哈哈一笑，毫不吝啬自己的夸奖。
　　殊不知，就在刚刚，一头熊妖诚惶诚恐的从这里飞速逃离。
　　“噼里啪啦！”
　　几乎在李念凡刚刚躲进山洞的同时，一道碗口粗的雷电便从天而降，落在了山林的某处！
　　原本陷入黑暗的世界瞬间被照得透亮。
　　大雨随之倾盆而下。
　　李念凡站在山洞口，向着远处的天际眺望。
　　“轰隆！”
　　居然又是一道雷电落下，这次的雷电比上次还要刚猛，如同一道银色的天柱一般，矗立在天地之间。
　　太恐怖了。

　　李念凡的眼眸微微一眯，整个人都愣住了。
